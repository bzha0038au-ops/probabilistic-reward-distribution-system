SDK Quickstart

Use the sandbox key to validate connectivity, reward request shape, and response handling.

1. Install SDK

npm install @reward/prize-engine-sdk

2. Configure Client

import { PrizeEngineClient } from '@reward/prize-engine-sdk';

const client = new PrizeEngineClient({
  apiKey: process.env.REWARD_ENGINE_API_KEY,
  baseUrl: 'https://api.rewardportal.io/v1/engine',
  environment: 'sandbox'
});

const reward = await client.rewards.create({
  projectId: 'proj_sandbox_01',
  subject: { id: 'user_99' },
  amount: 2500,
  currency: 'USD',
  metadata: { source: 'portal-quickstart' }
});

console.log(reward.id, reward.status);

Base URL

https://api.rewardportal.io/v1/engine

All requests must be made over HTTPS with a sandbox or production key scoped to the current tenant project.
